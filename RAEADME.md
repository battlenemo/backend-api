# Backend CRUD API (FastAPI)

這是一個使用 **FastAPI + SQLAlchemy + SQLite** 建立的後端 RESTful API 專案，
提供基本的 CRUD（Create / Read / Update / Delete）功能。

## 技術棧

- Python 3
- FastAPI
- SQLAlchemy ORM
- SQLite
- Uvicorn
- Swagger UI

## 功能說明

- 新增 Item 資料
- 查詢所有 Item
- 更新 Item
- 刪除 Item
- 使用 SQLite 進行資料持久化
- 自動產生 API 文件（Swagger UI）
